# Lab18-21
## База данных автомобилей

Консольное приложение для создания, хранения и управления информацией об автомобилях.

## Назначение
Проект демонстрирует практическое применение структур данных, файлового ввода-вывода, алгоритмов поиска и сортировки в языке C. Реализует систему управления базой данных автомобилей с возможностью выполнения условия: "Вывести сведения о машинах марки 'Ford' и зарегистрированных до 2000-го года".

## Функциональные возможности
Основной функционал:
✅ CRUD-операции: Создание, чтение, обновление записей об автомобилях

✅ Поиск: По марке автомобиля

✅ Сортировка: По году выпуска

✅ Фильтрация: Автоматический поиск Ford'ов, зарегистрированных до 2000 года

✅ Сохранение: Текстовый формат для долговременного хранения

✅ Консольный интерфейс: Интерактивное меню с 7 командами

## Структура данных:

## Реализованные функции:
fill_array() - заполнение массива случайными данными об автомобилях

search_material() - линейный поиск автомобиля по марке

compare() - функция сравнения для сортировки по году выпуска

load_file() / save_file() - работа с файлами

edit_one() - редактирование записи об автомобиле

print_array() - вывод данных с автоматической фильтрацией по условию задачи

## Сборка и запуск
Требования:
Компилятор C (GCC/Clang/MSVC)

Стандарт C99 или новее

Операционная система: Windows/Linux/macOS

# Запуск программы
Использование:
Запустите программу

Выберите действие из меню:

1 - Загрузить данные из файла

2 - Заполнить случайными данными (5 автомобилей)

3 - Показать все записи с автоматическим выводом Ford'ов до 2000 года

4 - Изменить запись по индексу

5 - Найти автомобиль по марке

6 - Сортировать по году выпуска

7 - Сохранить в файл

0 - Выход

Важно: Для выхода выберите пункт 0. Не забудьте сохранить данные перед выходом!
